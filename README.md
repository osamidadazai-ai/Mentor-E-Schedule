
# Mentor-E-Schedule

Mentor-E-Schedule is an AI-powered productivity and strategic planning platform built using Google AI Studio and Gemini.

## Features

- AI Mentor
- Strategic Task Planning
- Chess-based Priority System
- Daily Reflection
- Weekly & Monthly Reports
- Firebase Authentication
- Multi-device Synchronization
- Offline Support
- Analytics Dashboard

## Tech Stack

- Google AI Studio
- Gemini 2.5 / 3.5 Flash
- React
- TypeScript
- Vite
- Tailwind CSS
- Firebase Authentication
- Firestore
- Zustand
- Express.js
