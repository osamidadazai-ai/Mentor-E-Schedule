
# Mentor-E-Schedule

Mentor-E-Schedule is an AI-powered productivity and strategic planning platform built using Google AI Studio and Gemini.

## Features

- AI Mentor
- Strategic Task Planning
- Chess-based Priority System
- Daily Reflection
- Weekly & Monthly Reports
- Firebase Authentication
- Multi-device Synchronization
- Offline Support
- Analytics Dashboard

## Tech Stack

- Google AI Studio
- Gemini 2.5 / 3.5 Flash
- React
- TypeScript
- Vite
- Tailwind CSS
- Firebase Authentication
- Firestore
- Zustand
- Express.js

## Agent Capabilities

- **Goal Understanding:** Interprets user objectives, tasks, deadlines, and constraints to understand what needs to be achieved.

- **Intelligent Prioritization:** Evaluates tasks based on urgency, importance, dependencies, and expected impact to recommend what should be done first.

- **Adaptive Scheduling:** Generates practical schedules by balancing workload, available time, and upcoming commitments.

- **Personal Productivity Coaching:** Provides personalized suggestions, insights, and improvement strategies based on user activity and reflections.

- **Continuous Learning Loop:** Uses user feedback and reflection patterns to improve future recommendations and planning quality.

### Why It Is More Than a Reminder System

Traditional productivity apps only notify users about deadlines. Mentor E-Schedule actively assists users by helping them decide:
- What action should be taken next?
- Which tasks deserve priority?
- How should time be allocated?
- How can productivity habits be improved?

The AI agent transforms passive task management into proactive decision support.****
