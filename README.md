# Mentor-E-Schedule

Mentor-E-Schedule is an AI-powered productivity and strategic planning platform built using Google AI Studio and Gemini.

Unlike traditional productivity tools that only provide reminders, Mentor-E-Schedule acts as an intelligent productivity companion that helps users understand goals, prioritize tasks, create effective schedules, and continuously improve their productivity.

## Features

- 🤖 AI Mentor
- 📅 Strategic Task Planning
- ♟️ Chess-based Priority System
- 📝 Daily Reflection
- 📊 Weekly & Monthly Reports
- 🔐 Firebase Authentication
- 🔄 Multi-device Synchronization
- 📱 Offline Support
- 📈 Analytics Dashboard

## Tech Stack

- Google AI Studio
- Gemini 2.5 / 3.5 Flash
- React
- TypeScript
- Vite
- Tailwind CSS

- Firestore

- Express.js

## AI Agent Capabilities

## Goal Understanding
Interprets user objectives, tasks, deadlines, and constraints to understand what needs to be achieved.

## Intelligent Prioritization
Evaluates tasks based on urgency, importance, dependencies, and expected impact to recommend what should be done first.

## Adaptive Scheduling
Generates practical schedules by balancing workload, available time, deadlines, and user priorities.

## Personal Productivity Coaching
Provides personalized suggestions, insights, and improvement strategies based on user activity and reflection patterns.

## Continuous Improvement Loop
Uses user feedback and reflection data to improve future recommendations and planning quality.


## Why It Is More Than a Reminder System

Traditional productivity applications only notify users about deadlines.

Mentor-E-Schedule actively assists users by helping them answer:

- What action should be taken next?
- Which tasks deserve priority?
- How should available time be allocated?
- How can productivity habits be improved?

The AI agent transforms passive task management into proactive decision support.
